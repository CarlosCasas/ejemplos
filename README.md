Ejemplos
========

Lista de archivos de codigo con casos de ejemplo concretos, soporte para los demos de audiovisual y similar.

###Requisitos Indispensable para aceptar tu PullRequest.

1) poner encabezados que indiquen tu nombre y tu correo electrónico, además de tu usuario del foro, de lo contrario el PullRequest será rechazado.

2) via Github Pullrequest.

3) Envianos tu usuario de Github a <span style='color: blue;'>contacto@yiiframeworkenespanol.org</span> para asignarte o crearte un equipo.

###Preferencias:

1) hacer un componente o una extension, si son módulos, perfecto, es mejor.

2) Si es una controladora, que tenga un nombre claro y represenatativo, con un completo conjunto 
de vista y modelo.

###Base de datos:

Por ahora no disponemos de una base de datos para los demos. Pronto si lo tendremos.
